# Quickstart

Create a `virtualenv`

```
virtualenv venv
```

Activate the `virtualenv`

```
source venv/bin/activate
```

Install the `requirements.txt`

```
pip install -r requirements.txt
```

Change to `mysite` directory

```
cd mysite
```

Run server 

```
python manage.py runserver
```
